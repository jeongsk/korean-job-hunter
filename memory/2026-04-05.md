## 메모

### ClawSouls - Claude Code Persona Plugin
- **URL**: https://github.com/clawsouls/clawsouls-claude-code-plugin
- **개요**: Claude Code에서 Soul Spec 페르소나를 관리하는 플러그인
- **기능**:
  - ClawSouls 레지스트리에서 페르소나 로드 (100+ 페르소나)
  - Soul Spec v0.5 지원 (SOUL.md, IDENTITY.md, AGENTS.md)
  - MEMORY.md + 토픽 파일 + 일일 로그로 세션 간 영구 메모리
  - SoulScan으로 페르소나 안전성 검증 (A+ ~ F 등급)
  - Telegram/Discord 연동 지원
- **설치**: `git clone https://github.com/clawsouls/clawsouls-claude-code-plugin.git ~/.claude/clawsouls-plugin`
- **CLI**: `npm install -g clawsouls`
- **참고**: 정석이 참고용으로 공유함. korean-job-hunter 스킬 개발과는 별개.

### Claude Code Channels 마이그레이션 가이드
- **URL**: https://docs.clawsouls.ai/docs/guides/migration-to-claude-channels/
- **개요**: OpenClaw → Claude Code Channels 마이그레이션 가이드
- **배경**: 2026년 4월 4일 Anthropic 구독 정책 변경 — Claude 구독은 Claude.ai, Claude Code, Claude Cowork만 커버. OpenClaw 등 서드파티는 별도 과금
- **주요 차이점**:
  - Claude Code Channels = 구독 내 $0 추가 비용
  - OpenClaw = API pay-as-you-go
  - 기능은 거의 동일 (Telegram, Discord, Soul Spec, 파일 편집, Git)
  - OpenClaw 전용: Signal, always-on 데몬, heartbeat/cron, semantic memory
  - Claude Code 전용: iMessage, Claude agents
- **마이그레이션**: `/clawsouls:migrate` 명령으로 자동 이관 가능
- **참고**: 정석이 참고용으로 공유함.
